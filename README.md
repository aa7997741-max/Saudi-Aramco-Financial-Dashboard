# Saudi Aramco Financial Dashboard

## Project Overview
This project is an interactive Power BI dashboard that analyzes Saudi Aramco's financial performance from 2019 to 2025.

## Files Included
- Saudi-Aramco-Financial-Dashboard.pbix – Power BI dashboard
- Saudi-Aramco-Financial-Data.xlsx – Dataset
- Saudi-Aramco-Financial-Report.pdf – Project report
- Dashboard.png – Dashboard preview

## Tools Used
- Power BI
- Microsoft Excel

## Key Insights
- Revenue trends
- Net income analysis
- Oil price trends
- Financial performance comparison
